x-enterleave.js
===============
HTML attribute to add dynamic CSS modifier on classes

Dependency
----------
None

How to start 
------------ 
Add JS dependency
```html
<script src="../x-enterleave.min.js"></script>
```

At the end of DOM
```html
<script>am.start();</script>
```
